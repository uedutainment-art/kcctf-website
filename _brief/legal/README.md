# _brief/legal/ — 법적 텍스트

⚠️ **이 폴더의 3개 파일은 운영팀(흰곰)이 작성한 법적 텍스트로 채워져야 합니다.**

## 필요한 텍스트

현재 kcctf.org에 이미 있는 법적 페이지 텍스트를 그대로 가져와 주세요:

| 파일 | 출처 | 내용 |
|---|---|---|
| `terms.md` | https://kcctf.org/terms | 이용약관 |
| `privacy.md` | https://kcctf.org/privacy | 개인정보처리방침 |
| `refund.md` | https://kcctf.org/refund | 환불 규정 |

## 어떻게 가져오나요?

### 방법 A: 브라우저에서 직접 복사
1. https://kcctf.org/terms 접속
2. cmd+A (전체 선택) → cmd+C (복사)
3. `terms.md` 파일에 붙여넣기 (cmd+V)
4. privacy.md, refund.md도 같은 방식

### 방법 B: 운영팀에 원본 텍스트 요청
흰곰 또는 이종화 이사장에게 카톡:
```
이사장님, 홈페이지 리뉴얼 작업 중인데
이용약관·개인정보·환불 규정 원본 텍스트 부탁드려요.
(현재 사이트에 있는 그대로 사용할 예정)
```

## 형식

각 .md 파일 첫 줄에 제목, 본문은 마크다운으로:

```markdown
# 이용약관

## 제1조 (목적)

이 약관은 사단법인 춘천국제탱고페스티벌이...

## 제2조 (정의)

...
```

## 주의

- **임의로 작성하지 마세요** — 법적 책임 있는 텍스트입니다
- **수정·번역 금지** — 한국어 원문 그대로
- **영어 버전이 필요하면** 별도로 안내 (현재 사이트도 영어 약관 따로 있는지 확인 필요)
